Massive - the asm.js benchmark


Massive is a JavaScript benchmark that focuses on large, real-world C++ codebases compiled into asm.js using Emscripten. This is just one pattern of JavaScript among many on the web, but is very important for things like games, and unique in that the generated JavaScript is often very large and challenging for JavaScript VMs to optimize, which motivates a benchmark dedicated to it.
